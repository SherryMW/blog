## 修改器

::: tip

游戏运行时，你的角色属性（如生命值、弹药数、金钱数）会存放在内存中，修改器的核心功能是扫描和修改进程的内存数据

**单机**游戏不想肝局外成长可以使用修改器快速获得角色毕业体验

**网络**游戏使用修改器会触发风控，例如你真正的金钱数量记录在服务器的数据库里。你用修改器把你客户端显示的数字从100改成10000，但当你去商店买东西时，客户端会向服务器发送请求：“我要买这个价值5000的装备”服务器会检查它的记录，发现你只有100块，就会立刻触发风控，对你的账号进行违规处罚

:::

::: info Cheat Engine 使用教程

1. 启动游戏 

    运行你想要修改的**单机**游戏

2. 关联进程

    打开 Cheat Engine，点击左上角【选择要打开的进程】，找到并选中正在运行的游戏进程

3. 首次扫描

    假设游戏中当前金币为100，在 Cheat Engine 的数值框中输入“100”，点击【首次扫描】。左侧将列出所有匹配的内存地址

4. 筛选地址

    返回游戏，通过操作使金币数量发生变化（例如增加到200）。接着回到 Cheat Engine，在数值框中输入新的数值“200”，点击【下一次扫描】。重复此步骤，直到左侧列表中仅剩1–2个地址

5. 修改数值

    双击筛选出的地址，将其添加到下方地址列表。在列表中双击【值】一栏，输入你想要的数值。返回游戏查看效果，若未立即生效，可通过再次获取金币触发刷新

:::
