::: info 游戏汉化

[下载汉化包](https://img.sherry4869.com/blog/collection/game/awayout/3DMGAME-A.Way.Out.CHS.Patch.v1.1-3DM.rar)

Windows 10 或者 Windows 11 点击下载时可能会出现【无法下载-检测到病毒】的提示，需要打开电脑设置里的 Windows 安全中心 -> 病毒和威胁防护 -> 病毒和威胁防护设置 -> 管理设置 -> 实时保护 点击关闭。如果电脑安装了 McAfee 迈克菲的话直接打开该软件，选择 -> 我的保护产品 -> 实时扫描 -> 点击关闭。待汉化包下载成功后再打开实时保护设置

如果是在 Steam 平台里下载的话游戏默认存放在 `\steam\steamapps\common\AWayOut` 目录下，把汉化包到游戏安装目录下，双击运行 3DMGAME-A.Way.Out.CHS.Patch.v1.1-3DM.exe，点击汉化游戏即可

**闪退问题：**

编辑电脑系统环境变量，新增以下系统变量：

变量名：OPENSSL_ia32cap

变量值：~0x200000200000000

## 实时战略（RTS）

::: info RTS（Real-Time Strategy）玩法

游戏要求玩家在实时（非回合制）环境中管理资源、建造基地、训练军队，并通过战术和战略击败对手。其核心特征包括：

- 资源采集与经济系统：玩家需收集资源（如金币、木材、矿石）以建造设施和单位
- 基地建设：通过建造防御工事、科技建筑等扩展势力
- 多单位控制：指挥大规模军队，平衡兵种搭配（如步兵、坦克、空军）
- 科技树升级：通过研究解锁高级单位或能力
- 宏观与微观管理：既需全局战略（如扩张领土），也需微观操作（如单位走位）

:::

- [x] 星际争霸1 ✨
- [x] 红色警戒2：共和国之辉 ✨

## 实时战术（RTT）

::: info RTT（Real-Time Tactics）玩法

游戏聚焦小规模战术指挥，省略资源管理和基地建设，强调战场即时决策。核心特点包括：

- 纯战术对抗：玩家直接控制预设单位，专注地形、阵型和技能搭配 
- 无资源采集：单位通常不可补充或有限增援 
- 小规模战斗：典型战斗涉及几十个单位，而非上百 
- 环境互动：利用掩体、高地、伏击等战术元素 
- 任务导向：常见于单人战役，目标多为歼灭、防御或渗透

:::

- [ ] 埃里克斯霍姆：被偷走的梦

## 塔防（TD）

::: info TD（Tower Defense）玩法

- 固定路径：敌人沿预设路线移动，玩家需在路径旁建造防御塔阻击
- 资源管理：通过击杀敌人获得金币，用于建造/升级防御塔
- 波次进攻：敌人分波次来袭，难度逐步提升

:::
